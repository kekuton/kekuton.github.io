# Вопросы для двоих

Мини-приложение на HTML/CSS/JS для вопросов в паре.

## Что внутри

- выбор категории свайпом;
- 25 вопросов в каждой категории;
- чистый экран вопроса: карточка, счётчик, свайп вверх и свайп вниз назад;
- простое завершение категории без процентов и конфетти;
- мягкая тема фона под выбранную категорию;
- случайный порядок вопросов без повторов внутри раунда;
- подтверждение 18+ с сохранением выбора на устройстве;
- красивая скрытая категория, открывается 5 тапами по названию приложения;
- Telegram-вибрация при основных действиях;
- сжатые обложки категорий для быстрой загрузки.

## Файлы

- `index.html` — разметка приложения;
- `style.css` — подключает CSS-модули из `styles/`;
- `styles/` — актуальные стили;
- `js/` — логика приложения;
- `questions.json` — вопросы;
- `images/` — обложки категорий.

## Запуск

Можно открыть через GitHub Pages или любой статический хостинг.
Локально лучше запускать через простой сервер, например:

```bash
python3 -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```
