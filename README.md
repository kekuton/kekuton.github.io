# Вопросы для двоих

Чистая HTML/CSS/JS-версия для GitHub Pages или Telegram Mini App.

## Что внутри

- `index.html` — разметка приложения.
- `style.css` — точка входа для стилей, внутри только импорты.
- `styles/` — нормальная структура CSS по частям:
  - `base.css` — переменные, шрифт, базовые правила;
  - `layout.css` — фон, загрузка, шапка, общая сетка;
  - `categories.css` — экран выбора категорий;
  - `question-card.css` — экран вопроса и свайп-карточка;
  - `modal-completion.css` — 18+ модалка, ошибка, финальный экран;
  - `responsive.css` — адаптация под маленькие экраны.
- `questions.json` — вопросы по категориям, ровно до 25 вопросов на категорию.
- `js/core.js` — состояние, загрузка вопросов, роутинг, Telegram WebApp.
- `js/ui.js` — отрисовка категорий, вопросов и финального экрана.
- `js/game.js` — запуск категории и стабильные свайпы вопросов.
- `images/` — обложки категорий.

## Что вычищено

- старые настройки;
- избранное;
- история;
- достижения;
- блиц;
- premium;
- лишние скрытые кнопки и бейджи.

## Запуск

Можно открыть `index.html` через локальный сервер или загрузить файлы в GitHub Pages.

Для локальной проверки:

```bash
python3 -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```
