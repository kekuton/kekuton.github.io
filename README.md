# Couples App v3

Это новая собранная версия мини-приложения для пар.

Что есть внутри:
- новый мобильный UI
- категории + Premium категории
- Hard Talk
- свайпы как в Tinder
- обсуждение после вопроса
- AI-анализ пары
- share-card в PNG
- история, достижения, любимые вопросы
- вопрос дня
- анонимный режим
- онбординг
- локализация RU / EN
- комнаты и приглашение партнёра через ссылку/код

Важно про комнаты:
- из коробки реальная синхронизация работает через `BroadcastChannel` — это удобно для теста в одном браузере / нескольких вкладках
- для полноценной синхронизации между разными устройствами через интернет нужно подключить backend (например Supabase)

## Запуск локально

```bash
cd couples_app_v3_full
python -m http.server 8000
```

Открой:

```text
http://localhost:8000
```
