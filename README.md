# Вопросы для двоих

Статический Telegram Mini App на HTML/CSS/JS.

## Что внутри
- `index.html` — основной экран приложения.
- `questions.json` — вопросы по категориям.
- `js/` — логика приложения, свайпов, доступа 18+ и рендера.
- `styles/` — разделённые стили интерфейса.
- `images/` — обложки категорий.

## Как менять вопросы
Открой `questions.json` и измени массив нужной категории. Желательно держать 25 вопросов в каждой категории.

## Как поменять оплату/код 18+
Открой `js/access.js` и поменяй:
- `paymentUrl` — ссылка на оплату;
- `codes` — секретные коды доступа.

## Деплой на GitHub Pages
Загрузи файлы из архива в корень репозитория. `index.html` должен лежать в корне.
