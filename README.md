# Couple Flow v3

Что добавлено:
- комнаты с live-синхронизацией через `BroadcastChannel` + `localStorage` (между вкладками/окнами/браузерами на одном устройстве)
- ссылка-приглашение и вход по коду комнаты
- любимые вопросы
- рейтинг категорий
- вопрос дня
- demo Premium и premium-категории
- сохранение красивой share-card в PNG
- парные достижения
- AI-like анализ результатов на клиенте
- локальная + cloud-ready история
- анонимный режим
- онбординг
- локализация RU / EN

## Важно
Для **настоящей кросс-девайс синхронизации** и облачной истории между разными телефонами нужно подключить backend.
В `app.js` оставлен блок `CLOUD_CONFIG` — туда можно добавить параметры Supabase.
Без backend проект всё равно полностью работает как локальный Mini App.

## Запуск
```bash
python -m http.server 8000
```

Открыть:
```text
http://localhost:8000
```
