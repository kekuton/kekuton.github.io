# Вопросы для двоих — GitHub Pages build

Что добавлено:
- режим шаринга результата картинкой
- категория 18+ с подтверждением возраста
- комнаты для мультиплеера по коду/ссылке/QR
- PWA + полноэкранный режим
- Telegram Mini App hooks

## Deploy

1. Залей файлы в репозиторий.
2. В Settings → Pages включи GitHub Pages.
3. Source: Deploy from a branch → main / root.

## Как работает комната

Создай комнату, получи код, отправь код или ссылку другу.
У обоих откроется одинаковая категория/рандом с одинаковым порядком вопросов.
