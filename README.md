# Вопросы для двоих

Мини-приложение на HTML/CSS/JS для GitHub Pages или Telegram Mini App.

## Что внутри

- Вертикальная fullscreen-лента категорий.
- 25 вопросов в каждой категории.
- Случайный порядок вопросов без повторов внутри раунда.
- Свайп вправо — следующий вопрос.
- Свайп вниз — назад к категориям.
- Подпись «Продолжить X / 25», если категория была начата раньше.
- Мягкий старт категории перед первым вопросом.
- Красивый финальный экран без процентов и конфетти.
- 18+ закрыта секретным кодом доступа.
- Пасхалка: 5 тапов по названию приложения открывают скрытую категорию.
- Telegram haptic feedback, если приложение открыто в Telegram.
- Favicon / мини-логотип.

## Как работает доступ к 18+

Категория `18+` закрыта. При нажатии появляется окно:

1. кнопка «Купить доступ»;
2. поле для секретного кода;
3. кнопка «Открыть категорию».

После правильного кода доступ сохраняется на этом устройстве в `localStorage`.

Коды и ссылку на оплату можно менять в файле:

```text
js/access.js
```

Там есть блок:

```js
paymentUrl: 'https://yoomoney.ru/to/000000000000',
codes: [
  'LOVE-7K2P',
  'HOT-93QD',
  'PAIR-X8M1',
]
```

Замени `paymentUrl` на свою ссылку ЮMoney/ЮKassa и добавь свои коды.

Важно: такой способ подходит для простого теста. Код можно переслать другому человеку. Для настоящей защиты нужен backend.

## Файлы

- `index.html` — основная разметка.
- `style.css` — импорты актуальных CSS-файлов.
- `styles/` — стили по блокам.
- `js/` — логика приложения.
- `js/access.js` — платёжная ссылка и секретные коды.
- `questions.json` — вопросы.
- `images/` — обложки категорий.

## Запуск

Открой `index.html` через локальный сервер или загрузи проект на GitHub Pages.

Для локального запуска можно использовать:

```bash
python3 -m http.server 8080
```

Потом открыть:

```text
http://127.0.0.1:8080
```

## Telegram Mini App fullscreen

При запуске в Telegram приложение вызывает `WebApp.expand()` и, если клиент поддерживает, `WebApp.requestFullscreen()`. Высота интерфейса синхронизируется с Telegram `viewportHeight`, поэтому карточки занимают весь экран и не режутся системными панелями.
