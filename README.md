# Вопросы для двоих 2.0

Новая чистая версия проекта с нуля.

## Что внутри
- новый мобильный UI
- экран приветствия
- выбор категорий
- стартовый экран категории
- прохождение вопросов
- экран результата
- история игр в localStorage
- базовая поддержка Telegram Mini App
- защита от double-tap zoom

## Запуск локально

```bash
python -m http.server 8000
```

Открой:

```text
http://localhost:8000
```
