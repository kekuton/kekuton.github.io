# Вопросы для двоих — Mini App

Интерактивная игра-викторина для пар в формате Telegram Mini App. Узнавайте друг друга лучше, отвечая на интересные вопросы!

## 🎮 Описание

Приложение представляет собой викторину с вопросами для двоих. Идеально подходит для романтических вечеров, знакомств или просто весёлого времяпрепровождения.

## 🚀 Запуск локально

```bash
python -m http.server 8000
```

Затем откройте в браузере:

```
http://localhost:8000
```

## 📱 Установка как Telegram Mini App

1. Загрузите файлы проекта на хостинг (GitHub Pages, Vercel, Netlify или любой другой)
2. Создайте нового бота в [@BotFather](https://t.me/BotFather) или выберите существующего
3. Используйте команду `/newapp` для создания Mini App
4. Укажите ссылку на `index.html` вашего размещённого проекта
5. Получите прямую ссылку на приложение и делитесь с партнёром!

## 📁 Структура проекта

```
├── index.html          # Основная страница приложения
├── style.css           # Основные стили
├── questions.json      # База вопросов для викторины
├── manifest.webmanifest # PWA манифест
├── js/                 # JavaScript файлы
├── styles/             # Дополнительные стили
├── icons/              # Иконки приложения
└── images/             # Изображения
```

## ✨ Особенности

- 🎨 Адаптивный дизайн для мобильных устройств
- 📲 Поддержка PWA (Progressive Web App)
- 💬 Автоматическое отключение вертикальных свайпов в Telegram WebView
- 🎯 Большое количество вопросов в `questions.json`
